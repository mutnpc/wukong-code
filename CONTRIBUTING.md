# Contributing to Wukong CLI (Public)

Wukong CLI is a closed-source project. We do not accept code contributions.

## How you can help

- **Report bugs**: [github.com/mutnpc/wukong-cli/issues](https://github.com/mutnpc/wukong-cli/issues)
- **Request features**: open a GitHub issue describing your use case
- **Report security issues**: see [SECURITY.md](./SECURITY.md)

## Code of conduct

Please be respectful and constructive in all interactions.

## Questions?

Contact [support@wukong.today](mailto:support@wukong.today).
